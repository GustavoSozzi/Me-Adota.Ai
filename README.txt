2- tive que instalar o pacote express (eu usei o npm install express prisma @prisma/client dotenv bcryptjs jsonwebtoken)

3- criar o arquivo sql a ser implementado no MySql para criação do banco (pode ser expandido ou alterado)

4-alterar o arquivo .env quando for rodar na sua máquina

5- criei um servidor express, o express utiliza um sistema de middleware que permite adicionar funcionalidades à sua aplicação de forma modular. Isso inclui autenticação, manipulação de cookies, parsing de JSON, entre outros.
